# Tutorials folder
* Installation of `pycatrobin` only guarantees running `extract_from_gc_snr.pu` and `Welchs_t_test.py`.
* To run `fANOVA.py`, please additionally install `scikit-fda`.
  ``` bash
  pip install scikit-fda
  ```
* Welch's t-test and fANOVA codes will be incorporated into the main package in the near future.
